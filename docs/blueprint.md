# **App Name**: EduConnect Hub

## Core Features:

- Google-Authenticated Login: Implement secure user authentication using Google accounts, restricted to specified institutional email domains for both visitors and administrators.
- Admin Login with Password: A dedicated login flow for administrators requiring an additional password verification after initial email authentication.
- Visitor Check-in System: Allow visitors to provide their College Department and Reason for Visit as part of the check-in process.
- Real-time Admin Dashboard: Provide administrators with a dashboard displaying real-time visitor statistics, including interactive filters for Day, Week, Month, Department, Reason for Visit, and Visitor Classification.
- Admin User Management: Enable administrators to search for specific users by name and manage user access, including the ability to block users.
- Comprehensive Visit Logging: Automatically record each visitor's check-in with timestamps, user emails, and department categories to facilitate detailed historical filtering.
- AI-Powered Dean's Report Tool: An LLM-powered tool to generate insightful summary reports for the dean, highlighting busiest hours of the day and most common reasons for visits.

## Style Guidelines:

- Primary color: A mature and clear blue (#1F50A8), chosen for its association with trust, professionalism, and educational institutions, suitable for headlines and interactive elements.
- Background color: A very light, desaturated blue (#ECF0F4) providing a clean, airy canvas that enhances readability for detailed data and dashboards.
- Accent color: A vibrant teal (#2ECEED), analogous to the primary blue but significantly brighter and more saturated, ideal for call-to-action buttons, active states, and highlighting key information.
- Headline and body text font: 'Inter', a grotesque-style sans-serif for its modern, objective, and highly legible appearance, which is crucial for data-rich dashboards and reports.
- Utilize a consistent set of clean, geometric line icons that clearly represent their function within a professional and intuitive interface.
- Employ a responsive, card-based dashboard layout with clearly defined sections for statistics, filters, and management tools to ensure data readability and ease of navigation on various screen sizes.
- Incorporate subtle and swift transitions for dashboard updates, filter changes, and menu navigation, providing fluid user feedback without distracting from critical information.
